# ControlJusPlus

Canal oficial de distribuição e atualização do ControlJusPlus, aplicativo de gestão jurídica da ECM Advogados Associados.

## Download

O instalador mais recente está disponível na área de **Releases** deste repositório.

## Atualizações automáticas

O aplicativo consulta o arquivo `latest.json` publicado na release mais recente. Antes da instalação, o ControlJusPlus valida a assinatura Ed25519, o tamanho do arquivo e seu hash SHA-256.

## Segurança dos dados

Este repositório não contém bancos de dados, credenciais, chaves privadas, licenças emitidas ou informações de clientes. Os dados do aplicativo permanecem no computador ou na pasta de dados configurada pelo usuário.
