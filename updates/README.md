# Canal de atualizações

Diretório reservado para a documentação do atualizador do ControlJusPlus.

Os arquivos efetivamente consumidos pelo aplicativo são publicados como anexos das releases:

- `ControlJusPlus_vX.Y.Z_Setup.exe` — instalador da versão;
- `latest.json` — manifesto assinado contendo versão, URL, tamanho e SHA-256 do instalador.

Uma nova versão somente deve ser anunciada depois que o instalador e o manifesto assinado estiverem disponíveis na mesma release.
